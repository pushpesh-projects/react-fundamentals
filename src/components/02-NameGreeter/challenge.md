Challenge #2: Name Greeter (Complete)
Build a component that:

Has an input field where user can type their name
Shows "Hello, [name]!" below the input (updates as you type)
Has a "Clear" button that empties the input
BONUS: Show "Please enter your name" if the input is empty (instead of "Hello, !")

Expected behavior:

User types "John" → Shows "Hello, John!"
User clicks Clear → Input becomes empty
If empty → Shows "Please enter your name" (bonus)
